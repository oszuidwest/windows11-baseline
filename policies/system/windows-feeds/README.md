# Windows Widgets Policies

These policies disable the Windows 11 Widgets feature.

## Policies

### `disable-feeds-widgets.txt`

Disables the Windows 11 Widgets panel.

| Setting | Value | Description |
|---------|-------|-------------|
| `AllowNewsAndInterests` | 0 | Disables Windows 11 Widgets panel and news feed |

## Why

- Removes distracting news/widgets from taskbar
- Prevents widgets panel from appearing on hover or click
- Reduces system resource usage
- Keeps workstations focused on work tasks
